# Backlog

## Purpose
Tracks scoped work that is accepted into the project but not yet started.

## How To Use
- Add ready-to-pick tasks here.
- Move active work to `in-progress.md`.
- Move completed work to `done.md`.

## Template

### Intake Rules
- One bullet per task.
- Start with a concise action phrase.
- Link feature, ADR, or bug docs when they exist.
