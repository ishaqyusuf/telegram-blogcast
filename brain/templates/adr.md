# ADR Template

## Purpose
Reusable template for architecture decision records in `brain/decisions/`.

## How To Use
- Copy this into a dated or numbered ADR file.
- Keep the decision crisp and tie it to a real architectural consequence.
- Link related implementation PRs or feature docs when possible.
